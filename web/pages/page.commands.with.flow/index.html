Comandos para o gerenciamento de uma branch com o flow:

git flow feature start nome_branch (Iniciando uma branch com o git flow)

git flow feature publish nome_branch (publicar branch para que tenha uma cópia do repositório remoto, assim mais pessoas podem trabalhar nela)

git add . ou nome_arquivo (adicionando ao git)

git commit -m "alguma_mensagem" (escrevendo uma mensagem para o commit)

git flow feature finish nome_branch (finalizando e excluindo branch)
